Descripción

Descripción

La aplicación “Lista de Seguimiento” es un proyecto personal realizado por el equipo investigador. Tiene origen en la necesidad de sistematizar y facilitar la toma de datos en el ámbito educativo. Una de las etapas más importantes del proceso de enseñanza aprendizaje es la evaluación, cada docente presenta una planificación anual con los criterios de evaluación para cada asignatura o espacio curricular.


    Objetivo general:

  • Diseñar el sitio web de la aplicación para Android lista de seguimiento.

    • Objetivos específicos:

  • Adaptar la interfaz de usuario de la aplicación original para ejecutarla en los navegadores utilizando html.
  • Mejorar la experiencia de usuario agregando estilos y animaciones con CSS y Bootstrap.
  • Programar las funcionalidades propias de la aplicación móvil diseñadas originalmente con Java y Kotlin traduciéndolas al lenguaje de JavaScript y PHP.
